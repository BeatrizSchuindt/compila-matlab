## Pré-requisitos

Para gerar o parser, é necessário ter o ANTLR v4.

1.  Faça o download do arquivo `antlr-4.x.x-complete.jar` no [site oficial do ANTLR](https://www.antlr.org/download.html).
2.  Deixe o arquivo no diretório raiz do projeto
3.  Adicione tipo .jar ao seu arquivo `.gitignore` para que o `.jar` não seja enviado ao repositório.

```
# Arquivos .jar
*.jar
```
